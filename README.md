# privact-policy-iClock

本隐私政策解释了 iClock（以下简称“本应用”）如何处理您的个人信息。

1. 数据收集与使用
本应用是一个完全运行在您本地设备上的应用程序。我们不收集、不存储、不处理、也不共享您的任何个人数据、设备信息或使用习惯。本应用不需要连接互联网，您的所有数据均仅保存在您的设备本地。

2. 第三方服务
本应用未集成任何第三方数据统计（Analytics）或广告（Advertising）SDK。

3. 政策的更改
如果我们决定更改隐私政策，我们会在本页面更新这些更改。
